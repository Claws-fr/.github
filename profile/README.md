# Claws

**Première agence française d'agents IA autonomes.**

Nous installons, configurons et maintenons des agents [OpenClaw](https://openclaw.ai) pour des PME et entreprises qui veulent automatiser leurs opérations sans envoyer leurs données dans le cloud.

## Ce qu'on fait

Votre agent IA tourne sur votre machine. Il gère vos messages, votre agenda, vos tâches. Il travaille pendant que vous faites autre chose.
